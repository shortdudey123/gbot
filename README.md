gbot
====

Introduction
------------

IRC bot that can load custom command modules

Setup
-----

1. Close this repo
2. Modify gbot.py in the root of the repo to your liking
3. Run it

	python gbot.py

Note: no packages need to be installed.  Everything used is in stdlib.
